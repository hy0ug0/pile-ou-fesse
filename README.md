# 🔋🍑 Pile ou Fesse

Un site web fun pour tirer à "pile ou fesse" - la version décalée du classique "pile ou face" !

## 🎯 Concept

Au lieu de pile ou face, tu obtiens soit :
- 🔋 **Une pile**
- 🍑 **Une fesse**

## ✨ Fonctionnalités

- 🎲 Tirage aléatoire avec animation
- 📊 Statistiques persistantes (sauvegardées localement)
- 📱 Responsive (fonctionne sur mobile et desktop)
- ⌨️ Utilise la barre d'espace pour tirer

## 🛠️ Stack

- **React** - Framework JavaScript
- **TypeScript** - Typage fort
- **Vite** - Build tool
- **LocalStorage** - Sauvegarde des stats

## 🚀 Comment lancer

```bash
# Installe les dépendances
pnpm install

# Lance le serveur de développement
pnpm dev
```

Ouvre [http://localhost:5173](http://localhost:5173) dans ton navigateur.
